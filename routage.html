6. Analyse de la table de routage

Commande : show ip route

R 204.204.7.0/24 [120/1] via 199.6.13.2, 00:00:09, Serial0
C 219.17.100.0/24 is directly connected, Ethernet0

Interprétation :

  • R : Route apprise via le protocole RIP.
  • C : Réseau directement connecté à l'interface.
  • Serial0 : Type d'interface physique utilisée.
