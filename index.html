Le concept VLSM

Pourquoi utiliser le VLSM ?

Contrairement au subnetting classique, le VLSM permet d'utiliser des masques de longueurs différentes pour chaque sous-réseau.

  • ✅ Efficacité : On ne gaspille plus d'adresses.
  • ✅ Hiérarchie : Idéal pour les grands réseaux d'entreprise.
  • ✅ Agrégation : Permet de résumer les routes (Supernetting).

// Exemple de découpage /24

Réseau Global: 192.168.1.0/24


LAN A (60 hôtes) -> 192.168.1.0/26

LAN B (20 hôtes) -> 192.168.1.64/27

WAN (2 hôtes) -> 192.168.1.96/30
