4. Mécanisme du Subnetting

Emprunt de bits (Cas Classe B)

Pour un masque de 255.255.240.0 :

240 en décimal = 1111 0000 en binaire.
Nombre de bits empruntés : 4.
Nombre de sous-réseaux : 2^4 = 16.

Validité des IP (Masque /28)

Sur le réseau 192.168.33.0 /28 :

  • 192.168.33.15 : Invalide (Adresse de broadcast du premier bloc).
  • 192.168.33.16 : Invalide (Adresse réseau du deuxième bloc).
  • 192.168.33.17 : Valide (Première IP utilisable du bloc).
